개인정보 처리방침

본 개인정보 처리방침은 jbjb.r-e.kr(이하 "사이트")가 제공하는 서비스 (로그인, 채팅, URL 생성 등) 이용 과정에서 수집되는 이용자 정보와 그 처리 방식을 설명합니다. 사이트는 「개인정보보호법」 등을 준수합니다.

서비스 소개 및 목적

JBJB 서비스는 사용자가 Google 계정을 통해 간편하게 로그인하여 실시간 채팅, URL 단축 및 관리 등 다양한 웹 기반 유틸리티 기능을 이용할 수 있도록 제공하는 개인 프로젝트 기반의 웹 애플리케이션입니다. Google OAuth는 사용자 식별 및 서비스 이용 권한 부여를 위해 사용되며, 수집된 정보는 서비스 제공 목적 외에 절대 사용되지 않습니다.

1. 수집하는 개인정보 항목

  • 구글 로그인 정보
    • 구글 계정 이름(표시 이름)
    • 구글 계정 이메일(로그인을 위한 식별용, 별도 저장 안 할 수 있음)
  • 채팅 서비스(jbjb.r-e.kr/chat)에서 입력한 닉네임
  • URL 생성 서비스(url.jbjb.r-e.kr)에서 입력한 URL
    • 사용자가 직접 입력한 URL
    • 생성된 단축 URL 정보

2. 개인정보 수집 및 이용 목적

  • Google 계정 기반 로그인 기능 제공
  • 채팅 서비스 내 닉네임 표시 및 사용자 구분
  • URL 단축/저장 서비스 제공 및 기록 관리
  • 서비스 운영, 기능 제공 및 서비스 품질 개선

3. 개인정보 보유 및 이용 기간

사이트는 Firebase를 이용하여 데이터를 저장하며, 별도의 보관 기간을 두지 않습니다. 사용자의 요청 시 데이터는 즉시 삭제됩니다.

  • Google 로그인 정보: 계정 연결 해제 시 즉시 삭제
  • 채팅 닉네임: 사용자가 기록 삭제 요청 시 즉시 삭제
  • 입력 URL: 사용자가 직접 삭제 요청 시 즉시 삭제

4. 제3자 제공 및 위탁

사이트는 이용자의 개인정보를 외부에 판매하거나 제공하지 않습니다. 다만 아래 경우에만 예외적으로 제공될 수 있습니다:

  • 이용자가 사전에 동의한 경우
  • 법률에 따라 요청받은 경우(수사기관 등)

5. 쿠키(Cookie) 사용

사이트는 로그인 유지 등 서비스 제공을 위해 쿠키를 사용할 수 있습니다.

6. 아동의 개인정보 보호

사이트는 만 14세 미만 아동의 개인정보를 수집하지 않습니다.
